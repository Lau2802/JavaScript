Acerca de la web

Esta pagina web es una version de una biblioteca donde se puede ver la portada del libro como su sinopsis.

La idea nace de un proyecto para practicar el lenguaje de programacion javascript y en un futuro poder implementarle frameworks de javascript para que sea más moderna y funcional la pagina web.
